# 이력서 프로젝트

이 프로젝트는 Claude Code로 **자신만의 HTML 이력서**를 만들어보는 실습입니다.

## 폴더

- `templates/` — 참고용 샘플 이력서. 디자인/구조 레퍼런스로 사용.
- `resume.html` — 수강생이 Claude와 함께 만들어갈 결과물 (처음엔 없음).

## 작업 원칙

- 단일 HTML 파일에 inline CSS 유지. 브라우저로 바로 열어 미리보기 가능해야 함.
- A4 인쇄 스타일(`@media print`)은 유지 — PDF 저장 용도.
- `templates/` 는 레퍼런스. 수정하지 말 것. 수강생이 원하면 다른 스타일로 자유롭게 만들어도 됨.
